## Customer-Behavior-Marketing-Funnel-Analysis
# Project Overview

This project analyzes customer behavior and marketing funnel performance using Python and Pandas. The objective is to understand customer distribution across loyalty tiers, identify high-performing acquisition channels, analyze demographic trends, and generate actionable business insights.

The project simulates a real-world marketing analytics workflow where customer data is transformed into meaningful reports and visualizations to support business decision-making.

# 🎯 Business Objectives
- Analyze customer loyalty funnel (Bronze → Silver → Gold).

- Measure customer drop-offs between funnel stages.

- Evaluate acquisition channel effectiveness.

- Calculate Gold customer conversion rates by channel.

- Study customer demographics (gender, age, and country).

- Generate recommendations for marketing optimization.

# 🛠️ Technologies Used

- Python

- Pandas

- Matplotlib

- CSV Data Processing

- Data Visualization

# 📂 Dataset Features

The dataset contains customer information such as:

- Customer ID
  
- Age

- Gender

- Country

- Acquisition Channel

- Loyalty Tier (Bronze, Silver, Gold)

# 📈 Analysis Performed

## 1. Customer Funnel Analysis
Counts customers in each loyalty tier.

Visualizes progression from Bronze to Silver to Gold.

Identifies funnel drop-offs.

## 2. Acquisition Channel Performance
Calculates total customers acquired per channel.

Determines how many Gold customers each channel generates.

Computes Gold conversion rates.

## 3. Demographic Analysis
Gender distribution analysis.

Country-wise customer distribution.

Average customer age calculation.

## 4. Drop-Off Analysis

Measures customer losses between:

Bronze → Silver

Silver → Gold

## 5. Business Recommendations

Generates strategic recommendations based on customer behavior and conversion performance.
